# PokeCatalog

Este é um aplicativo web simples que permite pesquisar Pokémon e visualizar suas estatísticas.

![alt text](./readme/prof.png)

## Tecnologias

- Vue.js
- Vue Router
- SCSS
- Axios
- PokeAPI

## Instalação

```bash
yarn install
```

ou

```bash
npm install
```

## Execução

```bash
yarn dev
```

ou

```bash
npm run dev
```
